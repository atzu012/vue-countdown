# vue-countdown
Basic on vue-cli.

Started From today and setting until time with url. 

For Example:

Setting deadline time at 2022/12/31.

``http://vue-countdown/?date=20221231``

See the [demo](https://atzu012.github.io/vue-countdown/?date=20221231) online.
